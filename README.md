**AI Agent Fundamentals** คือแหล่งเรียนรู้แบบเปิด สำหรับทุกคนที่อยากเข้าใจ AI Agent อย่างเป็นระบบ เนื้อหาเป็นภาษาไทย อธิบายตั้งแต่พื้นฐานจนถึงแนวทางปฏิบัติจริง

AI Agent กำลังเปลี่ยนวิธีที่เราทำงานกับ AI — จากเครื่องมือที่ตอบคำถามอย่างเดียว ไปเป็นระบบที่สามารถวางแผน ใช้เครื่องมือ จดจำบริบท ตรวจสอบตัวเอง และทำงานร่วมกับมนุษย์ได้อย่างมีประสิทธิภาพ

ที่เก็บนี้เป็นส่วนหนึ่งของ [OpenThai](https://github.com/OpenThai) — ชุมชนการเรียนรู้และวิจัยด้าน AI สำหรับคนไทย

## แกนหลักที่คุณจะได้เรียนรู้

- **AI Agent** — ระบบที่ใช้ LLM เป็นสมอง วางแผน ใช้เครื่องมือ และเรียนรู้จากผลลัพธ์
- **Agentic Workflow** — รูปแบบการทำงานแบบวนรอบ: วางแผน → ลงมือ → สังเกต → ปรับปรุง
- **Tool-use / Function Calling** — การให้ Agent เรียกใช้เครื่องมือภายนอกอย่างปลอดภัย
- **Memory & Context** — การจัดการความจำเพื่อให้ Agent ทำงานต่อเนื่องและมีบริบท
- **Planning & Reflection** — การแตกงานเป็นขั้นตอนและตรวจสอบตัวเอง
- **Human-in-the-loop** — การออกแบบให้มนุษย์ตรวจสอบและอนุมัติในจุดสำคัญ
- **MCP** — โปรโตคอลเปิดสำหรับเชื่อมต่อ Agent กับเครื่องมือและข้อมูล
- **Multi-agent** — ระบบที่ Agent หลายตัวทำงานร่วมกัน
- **Evaluation & Verification** — วิธีวัดและยืนยันว่า Agent ทำงานถูกต้อง
