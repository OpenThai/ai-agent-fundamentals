## วิธีการเรียน

1. อ่านตามลำดับ — แต่ละบทต่อยอดจากบทก่อนหน้า
2. ทำความเข้าใจ concept ก่อนลงมือ coding
3. ทดลอง concept ผ่านตัวอย่างใน `examples/` (เมื่อมี)
4. ทบทวน glossary ทุกครั้งที่เจอศัพท์ไม่คุ้น

##  prerequisite

- มีประสบการณ์เขียนโปรแกรมบ้าง (ภาษาใดก็ได้)
- เข้าใจพื้นฐาน LLM (รู้ว่า ChatGPT/Claude คืออะไร)
- ไม่จำเป็นต้องมีประสบการณ์ AI Agent มาก่อน

## กิจกรรมเสนอแนะ

- จับคู่อ่านแล้วสรุปให้กันฟัง
- ลองออกแบบ Agent ง่ายๆ สำหรับงานที่ตัวเองทำอยู่
- ตั้งคำถาม "ถ้าใช้ Agent จะทำงานนี้ต่างยังไง"
