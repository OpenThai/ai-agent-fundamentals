# Roadmap — AI Agent Fundamentals

แผนงานพัฒนาที่เก็บความรู้ AI Agent Fundamentals สำหรับผู้เรียนไทย

---

## Phase 0 — Foundation ✅

- [x] สร้างโครงสร้างที่เก็บและเอกสารพื้นฐาน
- [x] กำหนดหลักการและแนวทางการเรียนรู้ (PRINCIPLES.md)
- [x] สร้างคำศัพท์และนิยามภาษาไทย (GLOSSARY.md)
- [ ] ตรวจทานภาษาไทยและความถูกต้องของเนื้อหา
- [ ] เพิ่มตัวอย่างโค้ดเบื้องต้นใน docs

## Phase 1 — Core Lessons

- [ ] เขียนเนื้อหาครบ 10 บทเรียนใน docs/
- [ ] เพิ่มภาพประกอบและไดอะแกรมในแต่ละบท
- [ ] สร้างแบบฝึกหัดท้ายบทสำหรับผู้เรียน
- [ ] เชื่อมโยงเอกสารเพื่อสร้างเส้นทางการเรียนรู้ที่สมบูรณ์
- [ ] ตรวจสอบความสอดคล้องของเนื้อหาข้ามบท

## Phase 2 — Examples

- [ ] สร้าง simple-agent ตัวอย่าง Agent ขั้นต่ำที่ทำงานได้จริง
- [ ] สร้าง tool-use-demo ตัวอย่าง Agent ที่เรียกใช้เครื่องมือ
- [ ] สร้าง memory-demo ตัวอย่าง Agent ที่มีหน่วยความจำ
- [ ] เขียนคำอธิบายวิธีรันและทดสอบแต่ละตัวอย่าง
- [ ] เพิ่มการทดสอบอัตโนมัติสำหรับตัวอย่างโค้ด

## Phase 3 — Practical Agent Patterns

- [ ] เพิ่มเนื้อหา Agent Pattern พื้นฐาน (ReAct, Plan-and-Execute, Reflection)
- [ ] ตัวอย่าง Agent สำหรับงานค้นหาข้อมูล (Research Agent)
- [ ] ตัวอย่าง Agent สำหรับเขียนและตรวจสอบโค้ด (Coding Agent)
- [ ] ตัวอย่าง Agent สำหรับช่วยตัดสินใจ (Recommendation Agent)
- [ ] แนวทางการเลือก Pattern ให้เหมาะกับงาน

## Phase 4 — Thai-first Agentic Workflow

- [ ] เพิ่มตัวอย่างการใช้งาน Agent กับภาษาไทยโดยเฉพาะ
- [ ] การจัดการ Tokenization และ Context Window สำหรับภาษาไทย
- [ ] การใช้ LLM ไทย (เช่น Typhoon, OpenThaiGPT) กับ Agent
- [ ] คู่มือการปรับแต่ง Prompt สำหรับภาษาไทย
- [ ] การจัดการกับภาษาผสม (ไทย-อังกฤษ) ใน Agent Workflow

## Phase 5 — Community and Contributions

- [ ] เขียนแนวทางการร่วมพัฒนา (CONTRIBUTING.md)
- [ ] เพิ่มเทมเพลตสำหรับ Issue และ Pull Request
- [ ] เปิดช่องทางอภิปรายและถามตอบ (Discussions)
- [ ] รวบรวม feedback จากผู้เรียนจริง
- [ ] จัดทำแผนการอัปเดตเนื้อหาตามเทคโนโลยีที่เปลี่ยนไป
